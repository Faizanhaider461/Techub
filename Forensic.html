Course Modules

MODULE 1 Introduction to Cyber Security & Digital Forensics

This module introduces the fundamentals of cyber security and the role of digital forensics in today‚Äôs digital world. You will explore types of cyber threats, cybercrime case studies, the importance of forensic investigations, and the need for security professionals in every organization.

MODULE 2 Networking & Security Essentials

Learn the building blocks of networking and how they apply to security. Topics include TCP/IP, OSI model, protocols, firewalls, VPNs, IDS/IPS systems, and packet analysis. You will also perform lab work on network monitoring and secure communication techniques.

MODULE 3 Ethical Hacking & Penetration Testing

This module covers ethical hacking methodology, from reconnaissance and scanning to exploitation. You will learn tools like Nmap, Nessus, and Metasploit for penetration testing. Practical labs will help you simulate attacks and assess vulnerabilities in systems.

MODULE 4 System, Web & Application Security

Dive into system-level security and application security. Topics include securing Windows & Linux OS, authentication bypass, SQL injection, Cross-Site Scripting (XSS), and cross-site request forgery. You will learn to detect and patch these vulnerabilities effectively.

MODULE 5 Malware Analysis & Incident Response

This module focuses on identifying, analyzing, and defending against malware. Learn different malware types, reverse engineering basics, sandbox analysis, and response planning. You will also practice real-life case scenarios to handle incidents and breaches.

MODULE 6 Cryptography & Data Protection

Understand the principles of cryptography and data security. Topics include symmetric/asymmetric encryption, hashing, digital signatures, certificates, and securing databases. You will apply encryption techniques in practical lab exercises.

MODULE 7 Digital Forensics & Evidence Handling

This is the core of digital forensics. Learn how to collect, preserve, and analyze digital evidence while maintaining a proper chain of custody. You will study file systems, log analysis, memory forensics, and how forensic reports are used in legal investigations.

MODULE 8 Cyber Laws, Compliance & Professional Ethics

Gain knowledge of international and local cyber laws, data protection regulations, and compliance standards (GDPR, ISO, PCI-DSS). This module ensures you understand the legal and ethical responsibilities of a cyber security and forensics expert.

MODULE 9 Tools & Advanced Technologies

Hands-on training with the most in-demand tools: Kali Linux, Wireshark, FTK Imager, Autopsy, Burp Suite, and forensic imaging tools. Learn automation in security using Python scripts and SIEM (Security Information and Event Management) platforms.

MODULE 10 Final Capstone Project & Career Preparation

In the final module, you will work on a capstone project where you perform a complete penetration test and forensic investigation in a simulated environment. Additionally, you will get guidance for certifications (CEH, CHFI, Security+), resume building, and interview preparation.
