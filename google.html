Course Modules

MODULE 1 UX Fundamentals

Introduction to human-centered design, UX principles, and understanding user needs through empathy and observation.

MODULE 2 Design Thinking

Learn problem-solving frameworks, ideation, and brainstorming to approach design challenges creatively.

MODULE 3 Wireframing

Sketch ideas on paper and digital tools. Create low-fidelity wireframes to outline structure and layout effectively.

MODULE 4 Prototyping

Transform wireframes into interactive prototypes. Build user flows to simulate real-world interaction.

MODULE 5 UI Design Principles

Focus on visual hierarchy, typography, colors, and grid systems to create visually appealing and accessible designs.

MODULE 6 Tools & Platforms

Hands-on training with Figma, Adobe XD, and other popular UI/UX tools for modern product design.

MODULE 7 Usability Testing

Conduct user testing sessions, analyze results, and refine designs based on real user feedback.

MODULE 8 Final Project & Career Prep

Create a complete app or website design from research to prototype. Add it to your portfolio and learn job/freelancing strategies.
